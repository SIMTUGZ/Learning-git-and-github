# Introduction to Git Command

This repository will show you how to use Git commands and how to use them effectively.
It covers only the basics of Git commands.
After learning these basic commands, you will gain a better understanding of Git.

## Getting Started with Git

### Step 1: Install Git

Download and install [Git](https://git-scm.com/install/).

### Step 2: Verify Installation

1. Open Git Bash.
2. Check if Git is properly installed:

```bash
git --version
```

3. You're ready to create your first Git repository!

### Step 3: Get Help

If you need help while using Git, use these commands:

| Command | Purpose |
|---------|---------|
| `git help <command>` | Get help on a specific command |
| `git config -h` | Get help on git config |
| `git help terminal` | Terminal help |
| `git help config` | Config help |
| `git log -h` | Learn how to use log command |

You can also visit [Git's official documentation](https://git-scm.com/doc).
