# Git Navigation Commands

### Basic Navigation Commands
These commands help you move through folders, check your location, and manage your terminal workspace efficiently.

Use the following commands to move around your files and folders in the terminal.

1. Clear the terminal screen:

        clear

2. List files and folders in the current directory:

        ls

3. Show the current working directory:

        pwd

4. Change to another directory:

        cd <folder-name>

5. Create a Git repository in the current folder:

        git init

6. Exit the terminal:

        exit
